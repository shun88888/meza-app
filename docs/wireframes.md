## 3. モバイル対応

### 3.1 レスポンシブ設計
- 最小幅: 320px
- 最大幅: 480px（モバイルファースト）
- セーフエリア対応
- タッチジェスチャー

### 3.2 ナビゲーション
- ボトムタブナビゲーション
- スワイプジェスチャー
- ハンバーガーメニュー（必要に応じて）

### 3.3 インタラクション
- タップ領域: 最小44px
- スワイプ: 左右ナビゲーション
- プルトゥリフレッシュ
- 長押しアクション

## 4. アクセシビリティ考慮

### 4.1 コントラスト
- 文字: 最小4.5:1
- UI要素: 最小3:1

### 4.2 フォーカス管理
- キーボード操作対応
- フォーカス表示
- タブオーダー

### 4.3 読み上げ対応
- セマンティックHTML
- alt属性
- aria-label
- スクリーンリーダー対応
